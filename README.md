# exif-viewer

EXIF viewer is a tool to display EXIF information from JPEG files.

## Demo
[EXIF viewer](https://code4fukui.github.io/exif-viewer/)

## Features
- Displays EXIF data from JPEG files
- Supports various EXIF data types
- Handles invalid or corrupted JPEG files gracefully

## Requirements
- [exif-js](https://github.com/taisukef/exif-js/) library

## Usage
1. Drag and drop a JPEG file onto the page
2. The EXIF data will be displayed in a table
